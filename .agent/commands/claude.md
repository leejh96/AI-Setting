# Command: Claude

## Description
Interact with Anthropic's Claude AI specific to this project context.

## Usage
- Interactive Mode: `/claude`
- Single Prompt: `/claude "Your prompt here"`
- With Model: `/claude -m sonnet "Your prompt"`

## Flags
- `-p` / `--print`: Print response and exit (non-interactive).
- `-m` / `--model`: Specify model (e.g., opu, sonnet, haiku).
