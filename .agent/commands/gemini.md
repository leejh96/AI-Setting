# Command: Gemini

## Description
Interact with Google's Gemini AI directly from the command line.

## Usage
- Interactive Mode: `/gemini`
- Single Prompt: `/gemini "Your prompt here"`
- With Model: `/gemini -m gemini-1.5-pro "Your prompt"`

## Flags
- `-p` / `--prompt`: Send a non-interactive prompt.
- `-m` / `--model`: Specify the model version.
